# 动态边框

背景知识： border-image 

.box {
  position: relative;
  width: 400px;
  height: 200px;
  margin: auto;
  border: 24px solid;
  border-image: url(https://skullctf.com/images/skull-border.svg);
  border-image-slice: 32;
  border-image-repeat: round;
}
