# 边框内圆角

背景知识： box-shadow , outline

哈哈哈
<div class="container">
  <div>
    <input v-model="checked" type="checkbox" id="pb" checked />
    <label for="pb">box-shadow</label>
  </div>
  <div class="content" :class="{ active: checked }">哈哈哈</div>
</div>

.container {
  display: flex;
  flex-direction: column;
  justify-content: center;
  align-items: center;
  padding: 60px 80px 80px;
  label {
    color: #000;
  }
  .content {
    padding: 12px;
    width: 200px;
    margin: 20px auto;
    text-align: center;
    background-color: #fff;
    box-shadow: 0 0 0 4px var(--color-primary);
    outline: 10px solid var(--color-primary);
  }
  .content.active {
    border-radius: 5px;
  }
}
