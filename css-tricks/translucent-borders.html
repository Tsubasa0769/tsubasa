# 半透明边框

背景知识： background-clip

默认情况下，背景的颜色会延伸至边框下层，这意味着我们设置的透明边框效果会被覆盖掉，在css3中，我们可以通过设置 background-clip:padding-box 来改变背景的默认行为，达到我们想要的效果。

哈哈哈
  <div class="container">
    <input type="checkbox" id="pb" checked />
    <label for="pb">padding-box(默认)</label>
    <div class="content">哈哈哈</div>
  </div>

.container {
  padding: 60px 80px 80px;
  background-color: var(--color-primary);
  label {
    color: #fff;
  }
  input[id='pb']:checked ~ .content {
    background-clip: padding-box;
  }
  .content {
    padding: 12px;
    margin: 20px auto;
    background-color: #fff;
    border: 10px solid rgba(255, 255, 255, 0.5);
  }
}
