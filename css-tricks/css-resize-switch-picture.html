# resize 实现图片切换预览功能

  • 使用css的resize的属性让元素能水平进行拉伸

    背景知识： resize

TIP

拉伸的元素改变宽度后，全父元素的宽度也进行改变，从而把图片展示出来
本来拉伸的元素只有右下角可以进行拉伸的
但通过transform: scaleY() 的方式变成全高度都能拉伸就很巧妙

WARNING

这种方式只能在PC端使用,移动端使用不了

DETAILS 
<div class="picA">
  <div class="picB">
    <div readonly class="resizeElement"></div>
  </div>
</div>

.picA {
  background-image: url(https://z3.ax1x.com/2021/08/17/fhJdpQ.png);
  background-size: cover;
  width: 100%;
  height: 340px;
  border: 5px solid #f0e5ab;
  border-radius: 3px;
  box-shadow: 0 0 1px #999, -2px 2px 3px rgba(0, 0, 0, 0.2);
  padding: 0;
  margin: 1rem auto;
  position: relative;
  overflow: hidden;
}
.picB {
  background-image: url(https://z3.ax1x.com/2021/08/17/fhJUfg.png);
  background-size: cover;
  height: 340px;
  position: absolute;
  top: 0;
  left: 0;
  min-width: 0;
  max-width: 100%;
  box-sizing: border-box;
}
.picB:before {
  content: '↔';
  position: absolute;
  background: rgba(0, 0, 0, 0.5);
  font-size: 16px;
  color: white;
  top: 0;
  right: 0;
  height: 100%;
  line-height: 340px;
}
.resizeElement {
  resize: horizontal;
  overflow: scroll;
  opacity: 0;
  position: relative;
  top: 50%;
  left: 0px;
  height: 15px;
  max-width: 100%;
  min-width: 15px;
  width: 0;
  cursor: move;
  transform: scaleY(35);
  transform-origin: center center;
}

# 通过JS方式实现

背景知识： resize

↔
DETAILS 
<div class="g-outer" ref="outer">
  <div class="g-inner" ref="inner">
    <div
      class="inner-resize"
      @mousedown="handleMouseDown"
      @touchstart="handleMouseDown"
      @touchmove.stop="handleMouseMove"
      @touchend.stop="handleMouseUp"
    >
      ↔
    </div>
  </div>
</div>

.g-outer {
  background-image: url(https://z3.ax1x.com/2021/08/17/fhJdpQ.png);
  background-size: cover;
  width: 100%;
  height: 340px;
  border: 5px solid #f0e5ab;
  border-radius: 3px;
  box-shadow: 0 0 1px #999, -2px 2px 3px rgba(0, 0, 0, 0.2);
  padding: 0;
  margin: 1rem auto;
  overflow: hidden;
}
.g-inner {
  position: relative;
  background-image: url(https://z3.ax1x.com/2021/08/17/fhJUfg.png);
  background-size: cover;
  width: 15px;
  height: 340px;
  min-width: 15px;
  max-width: 100%;
}
.g-inner .inner-resize {
  position: absolute;
  background: rgba(0, 0, 0, 0.5);
  font-size: 16px;
  color: white;
  top: 0;
  right: 0;
  height: 100%;
  line-height: 340px;
  user-select: none;
  cursor: move;
}

<script>
export default {
  name: 'ResizeSwitchPicture',
  data() {
    this.isMove = false
    this.srcX = 0
    this.minWidth = 15 // 内元素的最小宽度
    this.maxWidth = 0 // 内元素的最大宽度
    return {}
  },
  methods: {
    handleMouseDown(e) {
      window.addEventListener('mousemove', this.handleMouseMove)
      window.addEventListener('mouseup', this.handleMouseUp)
      this.isMove = true
      // 获取最大宽度
      this.maxWidth = this.$refs.outer.clientWidth
      if (e.type === 'touchstart') {
        // 记录点击时X的位置
        this.srcX = e.touches[0].pageX
        window.document.documentElement.style.overflow = 'hidden'
      } else {
        // 记录点击时X的位置
        this.srcX = e.pageX
      }

      // 记录当时内元素的宽度
      this.srcW = parseInt(
        window.getComputedStyle(this.$refs.inner, null)['width']
      )
    },
    handleMouseMove(e) {
      if (!this.isMove) return
      let offsetX = 0
      if (e.type === 'touchmove') {
        offsetX = e.touches[0].pageX - this.srcX
      } else {
        offsetX = e.pageX - this.srcX
      }
      // console.log(offsetX)
      let w = this.srcW + offsetX
      if (w <= this.minWidth) w = this.minWidth
      if (w >= this.maxWidth) w = this.maxWidth
      this.$refs.inner.style.width = w + 'px'
    },
    handleMouseUp() {
      this.isMove = false
      window.removeEventListener('mouseup', this.handleMouseUp)
      window.removeEventListener('mousemove', this.handleMouseMove)
      window.document.documentElement.style.overflow = 'auto'
    },
  },
}
</script>
